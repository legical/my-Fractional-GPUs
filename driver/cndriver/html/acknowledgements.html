makeself

The self-extracting archive (also known as the .run file) is generated using makeself.sh: http://www.megastep.org/makeself/

The version of makeself.sh used to create the .run is bundled within the .run file, and can retrieved by extracting the .run file's contents, e.g.:

$ sh NVIDIA-Linux-x86_64-515.65.01.run --extract-only
$ ls -l NVIDIA-Linux-x86_64-515.65.01/makeself.sh
